# 1. Requirements
- PHP 8.0+
- MySQL 5.7+ / MariaDB 10.4+
- Composer (for PHPMailer)
- Apache / Nginx with `mod_rewrite`

# 2. Install PHPMailer
```bash
cd /path/to/qcsim
composer install
```
